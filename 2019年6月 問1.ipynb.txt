    "\n",
    "$E[Y] = \\lambda_2 = 2$\n",
    "$V[Y] = \\lambda_2 = 2$\n",
    "\n",
    "$E[X+Y] = E[X] + E[Y]= 3 + 2 = 5$\n",
    "\n",

    "\n",

    "## (2)\n",
    "\n",
    "以下に対する、条件付き確率 $P(A|B) = \\frac{P(A\\cap B)}{P(B)}$ を求める。\n",
    "\n",
    "$A$：$X + Y = 4$ となる\n",
    "$B$ : $X$ は $x$ 回、$Y$ は $y$ 回起こる\n",
    "\n",
    "よって\n",
    "\n",
    "$A \\cap B$：$X$ は $x$ 回、$Y$ は $4-x$ 回起こる\n",
    "\n",
    "となる。\n",
    "\n",
    "ポアソン分布の公式は、以下の通りである。\n",
    "\n",
    "$P_p(x) = e^{-\\lambda}\\frac{\\lambda ^{x}}{x!}$\n",
    "\n",
    "また、ポアソン分布の再生性より、以下が成り立つ。\n",
    "\n",
    "$P(B) = P_p(X + Y) = e^{-(\\lambda_1 + \\lambda_2)}\\frac{(\\lambda_1 + \\lambda_2)^{(X+Y)}}{(X+Y)!} = e^{-(3 + 2)}\\frac{(3 + 2)^{(4)}}{(4)!} = e^{-5}\\frac{5^4}{4!}$\n",
    "\n",
    "そして\n",
    "\n",
    "$P(A \\cap B) = e^{-\\lambda_1}\\frac{\\lambda_1 ^{X}}{X!} \\cdot e^{-\\lambda_2}\\frac{\\lambda_2 ^{Y}}{Y!} = e^{-\\lambda_1}\\frac{\\lambda_1 ^{x}}{x!} \\cdot e^{-\\lambda_2}\\frac{\\lambda_2 ^{(4-x)}}{(4-x)!} = e^{-3}\\frac{3 ^{x}}{x!} \\cdot e^{-2}\\frac{2 ^{(4-x)}}{(4-x)!}$\n",
    "\n",
    "上記より\n",
    "\n",
    "$P(A|B) = \\frac{P(A\\cap B)}{P(B)} = \\frac{e^{-3}\\frac{3 ^{x}}{x!} \\cdot e^{-2}\\frac{2 ^{(4-x)}}{(4-x)!}}{e^{-5}\\frac{5^4}{4!}} = \\frac{4!}{x!(4-x)!} \\cdot \\frac{3^x \\cdot 2^{(4-x)}}{5^4}$\n",
    "$= {}_4 C_x \\cdot \\frac{3^x}{5^4} \\cdot \\frac{2^{(4-x)}}{5^4} = {}_4 C_x \\cdot \\frac{3^x}{5^x} \\cdot \\frac{2^{(4-x)}}{5^{(4-x)}} = {}_4 C_x \\cdot {(\\frac{3}{5})}^x \\cdot {(\\frac{2}{5})}^{(4-x)} = {}_4 C_x  (0.6)^x (0.4)^{(4-x)}$\n",
    "\n",
    "このとき、確率変数 $X$ は $B(4, 0.6)$ に従うので\n",
    "\n",
    "$E[X] = np = 4 \\cdot 0.6 = 2.4$\n",
    "\n",
    "よって\n",
    "\n",
    "平均が $2.4$ の二項分布に従う。\n"
